Class: ColumnFiltersContainer

ColumnFiltersContainer

An instance of ColumnFiltersContainer will display added column filters in a per column tab navigation view. Each display representation of a column filter has edit and remove controls that trigger events when clicked. Those events area captured in the parent ColumnFilters view.

Constructor

new ColumnFiltersContainer(options)

The ColumnFiltersContainer displays and facilitates the adding, editing and removing of column filters.
